✨ Авто-поля при новых ID во второй строке
💡 Пример: https://xxxx.gw.dockhost.net (без слеша в конце)

📌 Логика: первая строка листа игнорируется (заголовки). ВСЕ ячейки второй строки (ID колонок) превращаются в поля формы.
🚀 Если во второй строке появляется новый ID — после повторной загрузки поле автоматически добавится.
